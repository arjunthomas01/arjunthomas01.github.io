Project 2: CHN Analytics Twitter Verification Challenge

What I learned

Going into Data I/O I was shaky on my data cleaning and data manipulation skills. I recently learned how to use pandas and numpy. The event was the first time that I would put my newfound skills to the test. They came in very handy. I was able to successfully clean Twitter data and perform thorough exploratory data analysis. After cleaning and manipulating the data I learned how to make a correlation matrix heat map using matplotlib. I learned how go through text data (tweets) and identify keywords. My biggest takeaway from the event was the power of machine learning in data analysis. My teammate used a random forest model to identify the predictors of a verified user. Having little exposure to machine learning, I was amazed. I'm studying machine learning as soon as I can!

Improvements

  • Creating a more comprehensive set of keywords to search for in tweets
  • Fine tuning the random forest model to improve accuracy
  • Being able to explain the results of the correlation matrix better
  • Creating more eye-catching visualizations (using more Tableau)
