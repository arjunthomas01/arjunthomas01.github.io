Project 2: CHN Analytics Twitter Verification Challenge

My Experience

Going into the 2021 Data I/O I was shaky on my data cleaning and data manipulation skills. I had recently learned how to use the Pandas and NumPy Python libraries. The event was the first time that I would put my newfound skills to the test. Capitol Hill North Group (CHN) challenged teams to predict Twitter verification status by examining features such as likes, followers, and tweet contents. Using Pandas and NumPy, I was able to successfully clean and manipulate the data. I parsed through user tweets and identified frequencies of keywords. I also created data visualizations utilizing the Matplotlib and Seaborn Python libraries to help my team and the judges understand the dataset (see the linked presentation). The correlation matrix shown above is the visualization I'm most proud of. In addition to applying Python libraries, I was introduced to machine learning at Data I/O. I watched my teammate use a Random Forest model to identify the predictors of a verified user. Having little exposure to machine learning, I was amazed. I'm taking a data science-focused machine learning course as soon as I can!

Skills Acquired/Practiced

  • Cleaning data using Pandas and NumPy
  • Using Matplotlib and Seaborn
  • Parsing text using Python
  • Delivering a technical presentation

Improvements

  • Creating a more comprehensive set of keywords to search for in tweets
  • Fine tuning the random forest model to improve accuracy
  • Explaining the results of the correlation matrix in simpler terms
  • Creating more eye-catching visualizations (using more Tableau)
