Project 3: CapitalOne Financial Hack

My Experience

My team and I were planning on doing a data related challenge for Hack OHI/O. Unfourtunately, the organization running the challenge had difficulty getting their servers working properly. We had to leave our comfort zone: working with data. We shifted our plan of action. We were going to learn something completely new on the fly. None of us had experience in web development, but we decided to make a budget checker Chrome extension for the CapitalOne financial hack challenge. Our goal was to increase financial literacy and make it easier for e-shoppers to manage their spending. In the process of making this product I learned about the power of JavaScript in web development. The manifest, the foundation, for the Chrome Extension was entirely JavaScript code. We only used HTML and CSS to change the extension's visual aesthetics. JavaScript was a hard language to learn informally. It took me a significant amount of time to make a calculator that subtracts spending from a budget. I gained respect for web developers, and I want to delve deeper into JS so that I can create functional and aesthetic web applications.

Skills Accquired/Practiced
